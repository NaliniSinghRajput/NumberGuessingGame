# NumberGuessingGame
Hello again!, I made a game today, called the Number Guessing game!
I made an interactive python game , that let's users guess a number.
The number im using is a secret number , means it is an already stated number and not a random generated number.
Let's get into how I made this!:
1.) first i made a variable, that stores the value of the secret number. In this context I made it a ball,
since i want the user to guess the number of balls i have.
2.) Next i printed out some basic starting comment, that gives the user the description on what to do and what is happening.
3.) Next, this is the main body, i now used a while loop, under which I also added a variable x which is the input that we're going to take from the user. I also added a True after while, so that my loop works until the conditions are true.
4.) Next under the while loop, i made 3 conditions, that were, firstly, if the number guessed by the user is greater than than the secret number, then it should print out "The number is too high, try again". then I added an elif statement, changing the logic, stating that if the number is less than my secret number, it should print, "Too low, try again".
The last logic is that, if the number is equal to the secret number then, then it should print, "Yay, you're right!".
5.) Lastly i added a break condition at the end of this , so that if the number is equal to m scret number, the user has won and now it shouldn't continue the loops anymore and so break off.
That concludes my NumberGuessingGame!
